# sound
A graphic equalizer


* Get the sound
 * Start from a recording.
 * Look for a package which can read in a sound file.
* Decompress it
 * Need to read sound to know.
* Transform it
 * Bucket into frequency buckets with FFT.
* Display it
 * Write some code to push pixel.
